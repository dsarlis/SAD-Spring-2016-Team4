Microservices - Web Server

Overview

  • This project defines a simple web-application for accessing accounts and products data.
  • All the account and product information is fetched via two RESTful interfaces to Accounts and Products microservice.

The Demo

  • Eureka Dashboard: http://localhost:8761
  • Check applications registered: http://localhost:8761/eureka/apps/
  • Fetch account by number: /accounts/123456789
  • Fetch account by name: /accounts/owner/Keri
  • Fetch account by search Search
  • Fetch product by number: /products/123456789
  • Fetch product by name: /products/name/Beer
  • Fetch product by search Search
